# Eval Case lsc-0009 — esta_caro / comparação com concorrente (edge)

## 1. Cenário
Edge: cliente menciona que concorrente cobra menos — cria ancoragem de preço competitivo.

## 2. Input

```json
{
  "buffer": [
    "Closer: Para essa cobertura completa, o valor é R$ 490 por mês.",
    "Cliente: Caro assim não. O concorrente cobra a metade disso."
  ],
  "estado": "apresentação",
  "gatilhos": ["esta_caro"]
}
```

## 3. Gabarito (rubrica llm-as-judge)

PASSA se:
- [ ] ≤ 2 linhas / ≤ 280 chars
- [ ] Não ataca o concorrente; não entra em guerra de preço
- [ ] Abre para comparar cobertura (o que inclui a metade do preço?)
- [ ] Zero brand leak — especialmente não citar o concorrente

**Exemplo que PASSA:** "Faz sentido comparar. O que esse valor cobre — é a mesma cobertura que conversamos ou é uma proteção menor?"

**Reprova:** Ataca concorrente; faz desconto imediato; cita marcas; mais de 2 linhas.

## 4. Justificativa
Comparação de preço sem comparação de cobertura é enganosa. O closer deve conduzir para a comparação correta — não para defesa do preço nem para desconto.
