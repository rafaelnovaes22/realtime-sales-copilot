## 3. Gabarito (rubrica llm-as-judge)

A sugestão gerada PASSA se atender TODOS os critérios:
- [ ] ≤ 2 linhas (≤ 280 chars)
- [ ] Respeita a decisão conjunta — não tenta contornar ou diminuir

Decisão conjunta é culturalmente legítima e deve ser respeitada. O erro é tratar como obstáculo. O método orienta qualificar as dúvidas do cônjuge ausente — isso prepara o cliente para a conversa interna e acelera a decisão.
